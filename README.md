# AI 电商主图与文案生成工具

AI 驱动的电商图文生成工具，提供单条与批量生成能力，支持多风格选择，保存到素材库。

## 技术栈

- React 18
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- lucide-react (图标库)

## 项目路由

| 路由 | 页面功能 |
|------|--------|
| `/` | 首页 Dashboard，展示统计概览与最近活动 |
| `/products` | 商品任务列表页，管理所有任务 |
| `/products/new` | 商品图文生成工作台，填写信息并生成 |
| `/templates` | 模板库，浏览并管理模板收藏 |
| `/settings` | 设置页，配置默认生成规则 |

## 本地启动

```bash
npm install
npm run dev
```
访问 `http://localhost:3000` 即可。

## 构建与部署

```bash
npm run build
npm start
```

## 功能特性

1. **商品信息填写**：支持材质、尺寸、颜色等扩展字段，参考图上传
2. **图片生成功能**：支持多种风格选择，一键生成主图
3. **文案生成**：AI 辅助文案创作，支持重新生成
4. **批量导入**：支持批量创建商品任务
5. **模板库**：收藏常用模板，快速复用

## 测试入口

- 首页点击 "一键测试全流程" → 商品列表页 → 再次点击测试
- 商品列表页 "批量导入" → "一键测试全流程" → 查看进度条
- 商品工作台点击 "一键测试" → 自动填充示例数据并生成

## 环境变量

复制 `.env.example` 到 `.env.local` 并配置：
- 数据库连接字符串
- AI 服务 API 密钥
- 图片存储配置
